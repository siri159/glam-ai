# GLAM AI — AI Photo & Video Studio

> An AI-powered photo and video editing studio with generative filters, beauty tools, and viral content creation features.

Built for the [8x Social Contest](https://8xengineer.com) — June 2026.

## 🚀 Live Demo

Open `index.html` in any modern browser — no build step, no dependencies.

## ✨ Features

- **9 AI Filters** — Glamour, Golden Hour, Cinematic, Neon, Dreamy, Vintage, B&W, Rose + Original
- **Beauty Tools** — Skin smoothing, Glow, Warmth, Brightness with Natural / Glam / Editorial presets
- **Adjustments** — Brightness, Contrast, Saturation, Exposure, Highlights, Shadows
- **Text Overlays** — Bold, Script, Minimal, Display styles with color picker
- **Viral Analyzer** — AI score with Visual Appeal, Trend Match, Engagement metrics + tips
- **Platform Export** — Optimized crops for TikTok (9:16), Instagram (4:5), YouTube (16:9)
- **Drag & Drop Upload** — Upload any JPG/PNG/WEBP
- **PNG Export** — Download edited photos

## 🛠 Tech Stack

- Vanilla HTML / CSS / JavaScript
- HTML Canvas API for pixel-level image processing
- No frameworks, no build tools, no dependencies
